AWS Cloud Project: Weather Data Retrieval System

This Weather Data API project is a cloud-based application that allows users to select a specific date and retrieve weather information, including temperature, humidity, and condition. If no data exists for the chosen date, the application displays an appropriate error message, ensuring clarity for users.

It leverages AWS services to implement a serverless architecture: S3 hosts the website, DynamoDB stores the weather data, and API Gateway manages API requests. These requests trigger a Lambda function, which processes the input, fetches the relevant data from DynamoDB, and returns the results in a JSON response.
